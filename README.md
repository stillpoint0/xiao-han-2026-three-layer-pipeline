## Abstract
The Three-Layer Pipeline (TLP) wraps every LLM call in pre-execution (inhale/perceive), execution (hold/generate from reception), and post-execution (exhale/self-inspect) checkpoints. TLP restores presence as the default system behavior across all conversations.
